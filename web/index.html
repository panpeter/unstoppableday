Unstoppable Day

Decentralized calendar stored on the blockchain.

Unstoppable Day is the first decentralized calendar run on the blockchain. Owned and curated by the crypto community, where everyone can find legit NFT drops, ICOs, and other crypto-related events.

The project is being built in public. Source code is accessible freely on GitHub and you can follow the development at @piotrekwilczek.

unstoppableday.io is created by Piotr Wilczek and the code is publicly accessible on GitHub.
