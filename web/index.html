Unstoppable Day

Decentralized calendar stored on the blockchain.

Unstoppable Day is the first community-owned, decentralized calendar run on the blockchain.

The goal is to create a community-curated calendar where everyone can find legit NFT drops, ICOs, and other crypto-related events.

The project is being built in public and you can follow the development at @piotrekwilczek.

unstoppableday.io is created by Piotr Wilczek and the code is publicly accessible on GitHub.
